# jsTool
原生JS封装的通用函数库

##API

### ```jsTool.type(value)```
返回value的类型。
```javascript
jsTool.type('javascript'); 

// "string"


jsTool.type(123); 

// "number"
```

### ```jsTool.isNumber(value)```
```javascript
jsTool.isNumber(123); 

// true

jsTool.isNumber('str')

//false
```

### ```jsTool.isString(value)```
```javascript
jsTool.isString('I am Rcong'); 

// true
```
